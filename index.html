Ultimate Cloud OS for Data Center

[Open this slide in new window]

DogeOS is a distribution based on SmartOS and FIFO project. It is made to be the ultimate cloud OS for data center. Currently it has following features

  • 0 time of installation for Resource Node (a.k.a chunter node).
  • < 10min time of installation for Head Node (a.k.a fifo zone node), even without Internet access.
  • All indurstry proven features of SmartOS: ZFS, Dtrace, KVM, Zones and Crossbow.
  • Ready-to-use management console from FIFO.
  • Nearly 100% resource utilization of hardware.

How To Install

System Requirement

  • Same as the SmartOS. Find it here.

Get the ISO or USB IMG in left links

  • Burn the ISO to a DVD.
  • gunzip dogeos-usb.img.gz, and write USB img to a USB drive > 2GB.
  • Then boot your server with live DVD or USB just like SmartOS.

Resource Node (a.k.a chunter node)

  • NO NEED! After the live system started, it works!

Head Node (a.k.a FiFo zone node)

  • As root, in bash key in /dogeos/bin/headnode-install, and enter
  • Follow the instructions
  • Then you are done!
