DogeOS is made to be ...?


the Ulitimate Cloud OS
for Data Center

Ultimate Cloud OS should ...


  • Do OS virtualization.
  • Do Hardware virtualization.
  • Do Network virtualization.
  • Be Storage hypervisor.
  • Have web portal and API for management work.
  • Have advance diagnostic tool for performance tuning work.

(DevOps in) Data Center wish ...


  • The OS occupies as less resources as possible, so most resources are left to tenants.
  • The OS is easy to install, and best if no installation is required.
  • The OS is easy to upgrade, and best if it is automatic.
  • The OS is able to install even without Internet access.

DogeOS fits ALL !


  • DogeOS is based on SmartOS → a solid reliable foudation.
  • DogeOS equips Project FiFo → the solid management layer with web portal and API.
  • DogeOS is made to be Live OS → no installation and auto upgrade.
  • DogeOS boots smoothly even without Internet access.
